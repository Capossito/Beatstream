Beatstream is your own music streaming service.

Beatstream is a free, web-based media server for listening to your music from your computer using a web browser. You only need to install Beatstream on your computer and set up the access to it. Beatstream is known to sometimes run & actually play music on Google Chrome, Firefox, iPad and iPhone.

See demo

P.S. To take full use of Beatstream, you should have a computer running Beatstream 24/7.

Installation

Beatstream currently runs on Ruby on Rails and its installation is a lengthy and difficult process. We are working on making it better, but that won't happen until Beatstream itself is running better. Nevertheless, if you are a bad enough dude, here it is:

Reveal the secrets ▼
Windows Mac Ubuntu/other

First, install Ruby 1.8.7, and remember to check the "Add Ruby executables to your PATH" checkbox while installing.

Second, install Bundler by running:

gem install bundler

Third, clone Beatsream with Git or Download Beatstream as a ZIP and extract.

Four. Set-up Beatstream by doing this stuff in command prompt (cmd):

cd Beatstream
bundle install
rake db:migrate
notepad config/musicpaths.yml
-> Change 'music_path'
bundle exec rails console
-> User.create(:username => 'your_name', :email => 'your_email@example.com', :password => 'asdf', :password_confirmation => 'asdf')
-> exit
bundle exec rails server -d

First, you need to have Ruby 1.8.7, RubyGems and sqlite3 development headers installed. All of these can be installed by installing Xcode and the command-line tools.

Second, install Bundler by running:

$ sudo gem install bundler

Third, clone Beatsream with Git or Download Beatstream as a ZIP and extract.

Current quirks

  • Last.fm: You need to manually go to https://<your_server>/settings and do the authentication there.
  • Media Library: Opens a blank page after refresh is done. Just go back, and start listening :)
  • Flash (Firefox): Beatstream's Flash plugin (SoundManager2) crashes sometimes. Restart browser / Flash.
  • UTF-8: Sometimes song info looks garbled. Sometimes Beastream can't play songs with funny characters in filenames or paths.
  • Endlessly spinning loader: You prorably have a Flash block of some kind. Allow Beatstream to use Flash.
