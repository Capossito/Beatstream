Beatstream

Your own music streaming service.
Listen to your music anywhere.

What?

Beatstream is a free, web-based media server for streaming your music from your computer to anywhere with a browser. You only need to install it on your computer and set up the access to it. Currently Beatstream runs reliably on Google Chrome and Firefox on the desktop. Also, currently in development and open-source!

To use Beatstream you need a computer running Beatstream somewhere all the time.

Check out the demo!

I also wrote a short blog post where I am announcing Beatstream and tell a little bit about developing Beatstream.

Installation

Installation is currently a bit icky , but we are working to make it smoother! Pinky swear!

Requirements: Ruby 1.8.7 or newer, RubyGems, bundler and sqlite3 development files

Windows Mac / OSX Ubuntu Other
Attention! Windows installation is currently borked!

Installing Beatstream on Windows is currently a tricky business and this guide does not work.
We are working to fix this. Stay tuned!

Step 1: Requirements

Install Ruby 1.8.7
from RubyInstaller (e.g. Ruby 1.8.7-p358).
Check the "Add Ruby executables to your PATH" checkbox while installing.

Install Bundler:
Open command prompt: Start → type cmd.exe .
Type into command prompt: 

gem install bundler

Step 2: Get Beatstream

Clone Beatsream with Git – or – Download Beatstream as a ZIP and extract.

Step 3: Set-up Beatstream

Do this stuff in command prompt:

cd Beatstream
bundle install
rake db:migrate
notepad config/musicpaths.yml
-> Change 'music_path'
bundle exec rails console
-> User.create(:username => 'your_name', :email => 'your_email@example.com', :password => 'asdf', :password_confirmation => 'asdf')
-> exit
rails server -d
